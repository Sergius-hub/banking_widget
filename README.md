### Использование функций
#### Функция маскировки номера карты
```Python
def get_mask_card_number(card_number: int) -> str:
    ...
    return card_number_mask
```
Функция принимает на вход номер карты и возвращает ее маску. 
Номер карты замаскирован и отображается в формате XXXX XX** **** XXXX, 
где X — это цифра номера. То есть видны первые 6 цифр и последние 4 цифры, 
остальные символы отображаются звездочками, номер разбит по блокам по 4 цифры, разделенным пробелами. 
Пример работы функции:
```Python
masked_card_number = get_mask_card_number(7000792289606361) # вызов функции
print(masked_card_number) # masked_card_number: 7000 79** **** 6361  
```
#### Функция маскировки номера счёта
```Python
def get_mask_account(account_number: int) -> str:
    ...
    return account_number_mask
```
Функция принимает номер счета на вход и возвращает его маску. 
Номер счета замаскирован и отображается в формате **XXXX, 
где X — это цифра номера. То есть видны только последние 4 цифры номера, 
а перед ними — две звездочки. 
Пример работы функции:
```Python
masked_account_number = get_mask_account(73654108430135874305)  # вызов функции
print(masked_account_number) # **4305 вывод функции
```
#### Функция маскировки номера счета или номера карты
```Python
def mask_account_card(account_or_card: str) -> str:
    ...
    return masked_number
```
Функция mask_account_card(account_or_card), принимает строку с номером карты 
или номером счёта, возвращает такую же строку только с замаскированной строкой 
номера карты или счёта. 
Пример работы функции:
```Python
mask_number_card = mask_account_card("Visa Platinum 7000792289606361")
mask_number_account = mask_account_card("Счет 73654108430135874305")
print(mask_number_card) # Вернёт: Visa Platinum 7000 79** **** 6361
print(mask_number_account) # Вернёт: Счет **4305
```
